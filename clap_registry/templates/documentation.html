📘 Clap Documentation

Everything you need to know to get started, manage your packages, and build C++ projects with Clap.

🔰 Getting Started

Install Clap globally using executable:

Download latest version

Verify with:

clap --help

📦 Create a New Project

Generate a boilerplate C++ project:

clap init

Sets up folders and a configured CMakeLists.txt.

📥 Install Packages

Add a package to your project:

clap install spdlog

Specify a version if needed:

clap install spdlog --version 1.15.3

Install all missing packages in clap_package.json

clap install

📃 List Installed Packages

Display all installed packages:

clap list

⚙️ Build & Run

Compile your project:

clap build

Execute the built binary:

clap run

🧹 Uninstall / Update

Remove a package:

clap uninstall glm

Update a package:

clap update fmt

Update all packages:

clap update --all

📁 File Structure Overview

  • packages/ - third-party libraries
  • src/ - your C++ sources
  • include/ - your headers
  • CMakeLists.txt - auto-managed build file
